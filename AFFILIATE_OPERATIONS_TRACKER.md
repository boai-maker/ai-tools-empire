## TECHNICAL REFERENCE

### How the affiliate system works

**Single source of truth:** `affiliate/links.py` — `AFFILIATE_PROGRAMS` dict
**ID storage:** `.env` → `config.py` → `AFFILIATE_IDS` dict
**Tracking redirect:** `/go/{tool_key}` in `main.py` — reads env live, logs click to DB
**Sidebar CTA:** driven by `featured_tool` column in `articles` DB table
**Click analytics:** visible at `/admin` dashboard → affiliate clicks table

### To activate a new affiliate ID:
1. Add to `.env`: `TOOL_AFFILIATE_ID=your_id`
2. No restart needed — `/go/{tool_key}` reads live
3. If the ID is a full URL (like ElevenLabs), paste the full URL — the route handles it

### To add a new program completely:
1. Add entry to `AFFILIATE_PROGRAMS` in `affiliate/links.py`
2. Add key to `AFFILIATE_IDS` in `config.py`
3. Add entry to `aff_url_map` and `fallback_map` in `main.py` `/go/` route
4. Add env var to `.env.example`
