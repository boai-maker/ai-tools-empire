## 1. The Audit Prompt

Copy this verbatim into Claude or ChatGPT (works best with Claude Sonnet 4.5):

```
You are a senior AI tooling consultant auditing a personal/business AI stack.
The user pastes a list of every AI tool they pay for or use heavily.

Output structure (no markdown headers, plain prose):

- 3 things that are working well (be specific — what's the tool earning them
  in time saved or revenue?)
- 2 tools they should drop (cite the cheaper or free alternative)
- 1 tool they should swap (the upgrade and why)
- 3 workflows they should add (concrete, executable in 1 hour each)

Tone: dry, sharp, founder-to-founder. No "I hope this helps." No bullet
points in the prose. Maximum 250 words.

Stack to audit:
[paste the user's tools here, one per line, with how often they're used]
```

## 2. The Scoring Rubric

For each tool in the stack, score on three axes (1-5 each):

| Axis | What to ask |
|---|---|
| **Time saved per week** | How many hours does this save vs doing it manually? Below 1hr → drop |
| **Revenue or output tied to it** | Has it directly produced a sale, lead, or shippable artifact this month? Zero → drop |
| **Switching cost to alternative** | If a free or cheaper tool exists, how painful is the migration? High pain might justify keeping |

**Decision rule:** keep if total ≥ 9, swap if 6-8, drop if ≤ 5.

## 5. The 7-Day Follow-up Sequence

### Day 1 (immediately after audit delivered)
Subject: One thing to do today
> Quick favor: pick the easiest "drop" from the audit (usually the cheapest
> tool you're paying for) and cancel it right now while it's top of mind.
> You'll be surprised how many tools auto-renew because you forget. Reply
> "done" and I'll send you next week's playbook.

### Day 3
Subject: How's the new stack feeling?
> Just checking in. Did you cancel any of the tools we flagged? What's been
> the trickiest swap? Hit reply, I'll send a 5-line response with my best
> answer.

### Day 7
Subject: The bigger question
> A week in — what's the one bottleneck you wish AI could solve but no tool
> exists yet? The audit covered what to drop. The bigger lift is what to
> ADD. If you tell me, I'll spend an hour mapping a custom workflow for $97.
